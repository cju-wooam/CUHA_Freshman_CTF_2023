# CUHA Freshman CTF 2023
2023년도 청주대학교 디지털보안학과 1학년 대상 CTF

# How to use
1. npm i 이후 node app.js 명령어를 통해서 실행할 수 있음(DB 필요)
2. 회원가입 후 로그인하면 문제를 볼 수 있음
3. Scoreboard, Profile, Settings, Rules 페이지를 이용할 수 있음
